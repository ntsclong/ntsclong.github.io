vue-devtools安装

vue 

由于vue是数据驱动的，所以这就存在在开发调试中查看DOM结构并不能解析出什么。
但是借助vue-devtools插件，我们就可以很容易的对数据结构进行解析和调试。

这里介绍vue-devtools的安装方法：

第一种方法

直接下载crx文件进行安装。点此下载

  1. 下载完成后将crx文件拖入谷歌浏览器——更多工具——扩展程序中
  2. 找到目录
    C:\Users\Administrator\AppData\Local\Google\Chrome\User Data\Default\Extensions\nhdogjmejiglipccpnnnanhbledajbpd\3.1.2_0
    将其中manifest.json文件里的"persistent"参数修改为true。

第二种方法

GitHub地址：
https://github.com/vuejs/vue-devtools
克隆git项目

  1. 1git clone https://github.com/vuejs/vue-devtools.git 


完成后进入项目进行依赖安装，你可以

  1. 1npm install 
  2. 2//npm可能有点慢，建议安装cnpm进行安装 
  3. 3cnpm install 

安装完成后build一下项目

  1. 1npm run build 


到这里，你再打开vue-devtools目录下的 \shells\chrome 目录，打开manifest.json文件
将其中的persistent修改为true

最后就是将工具安装到chrome中

  1. 打开chrome浏览器，打开设置>点击或者程序>点击开发者模式
  2. 再点击加载已解压的扩展程序，选择vue-devtools下的 \shells\chrome 目录，确定。
    注意：这里的vue-devtools目录不可删除

安装完成，可以愉快的用vue-devtools进行开发了
