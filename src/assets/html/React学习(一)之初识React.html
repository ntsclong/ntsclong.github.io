React学习(一)之初识React

React

React是什么？

  • React是一个用户构建用户界面的JavaScript库。
  • React主要用于构建UI，很多人认为React是MVC中的V（视图）。
  • React起源于Facebook的内部项目，用来架设 Instagram 的网站，并于 2013 年 5 月开源。
  • React 拥有较高的性能，代码逻辑非常简单，越来越多的人已开始关注和使用它。

React的特点

  1. 声明式设计 - React采用声明范式，可以轻松描述应用。
  2. 高效 - React通过对DOM的模拟，最大限度地减少与DOM的交互。
  3. 灵活 − React可以与已知的库或框架很好地配合。
  4. JSX − JSX 是 JavaScript 语法的扩展。React 开发不一定使用 JSX ，但我们建议使用它。
  5. 组件 − 通过 React 构建组件，使得代码更加容易得到复用，能够很好的应用在大项目的开发中。
  6. 单向响应的数据流 − React 实现了单向响应的数据流，从而减少了重复代码，这也是它为什么比传统数据绑定更简单。
React高效原因
1、 虚拟（virtual）DOM，不是总是直接操作DOM
2、 高效的DOM Diff算法，最小化页面重绘

React几个重要概念

  1. 模块与组件
    • 模块：
      • 理解：向外提供特定功能的js，一般就是一个js文件
      • 作用：简化js的编写，阅读，提高运行效率
    • 组件：
      • 理解：用来实现特定功能效果的代码集合（html/css/js）
      • 作用：复用，简化项目编码，提高运行效率
  2. 模块化与组件化
    • 模块化：当应用的js都以模块化的概念来编写的，这个应用就是模块化的应用
    • 组件化：当应用是以多组件的方式来实现功能的，那这个应用就是一个组件化的应用

如果一个应用使用模块化和组件化去进行开发，那么这个项目又叫工程化项目

  • react.js：React的核心库
  • react-dom.js：提供DOM的扩展库(虚拟DOM)
  • babel.min.js：解析JSX语法代码专为纯JS语法代码的库

虚拟DOM

提供了一些AP来创建一种“特别”的js对象

    let user =React.createElement(
        'h2',//html标签名（h1,span）
        {id:'myTitle'},//标签属性（id,class）
        'hello React!!!'//写入的内容
    );
    //等价于
    let user = <h2 id="myTitle">hello React!!!</h2>;
		
    //以上两种方法都可以用于创建虚拟DOM
    ReactDOM.render(user, document.getElementById('container'));

创建的虚拟DOM最终会被React转换为真实DOM，我们一般操作的是虚拟DOM

JSX

全称：JavaScript XML，React定义的一种类似XML的JS扩展语法（XML+JS）
作用：用来创建React虚拟DOM（元素）对象

    let ele = <h1>hello JSX!!!</h1>;
    //等价于
    let msg = 'hello JSX!!!';
    let ele = <h1>{msg}</h1>;

最终创建的JSX，它不是字符串，也不是一个HTML/XML标签；而是一个JS对象
语法法则

  • 遇到<开头的代码以标签解析
  • 遇到{开头的代码，以JS语法解析；标签中的JS代码必须用{}包裹

注意：如果需要执行JSX语法，script标签需要将type属性设置为：text/babel

  1. 1 let names = ['Tom', 'Jack', 'Bob']; 
  2. 2 ReactDOM.render( 
  3. 3 <ul> 
  4. 4 {names} 
  5. 5 </ul>, 
  6. 6 document.getElementById('example1') 
  7. 7 ) 

JSX会自动遍历数组中的所有元素，所以以上代码会在ul中输出数组的每一项，如果需要将数组以列表进行渲染，需要进行map遍历并返回

  1. 1 let names = ['Tom', 'Jack', 'Bob']; 
  2. 2 ReactDOM.render( 
  3. 3 <ul> 
  4. 4 { 
  5. 5 names.map((item, index) => { 
  6. 6 return <li key={index}>{item}</li> 
  7. 7 }) 
  8. 8 } 
  9. 9 </ul>, 
  10. 10 document.getElementById('example1') 
  11. 11 ) 
  12. 12 //key属性如果不加会出现警告（a key should be provided for list items）意思是当你创建一个元素时，必须包括一个特殊的key属性。
